### Python函数的参数传递

Python中的类型是属于对象的，而非变量，Python中的对象有两种：<br/>
- 可更改(mutable)对象,包含int、float、strings, tuples、numbers等等。
- 不可更改(immutable)对象，包含list、dict等等。

### 数据结构
- List(列表)：序列结构，可以被修改
- Tuples(元组)：序列结构，不可以被修改；
- Dict(字典)
- Set(集合)：元素不重合且无序，不支持索引。
