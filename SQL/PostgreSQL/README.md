## PostgreSQL

对象-关系型数据库

PostgreSQL是客户端/服务器模式，客户端和服务器通过TCP/IP通信。Postgre服务器可以处理客户端的多个并发请求，它为每个连接fork一个新的进程，之后客户端和新服务器进程便不再经过最初的Postgre进程的干涉进行通讯。
